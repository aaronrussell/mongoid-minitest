# mongoid-minitest

MiniTest matchers for Mongoid.

**First release:** 1 week! i hope :grin:

## Installation

Add this line to your application's Gemfile:

    gem 'mongoid-minitest'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mongoid-minitest
